## Задача 1. Исследование мультипликативной структуры $GF(5^2)$

Требуется определить порядки всех элементов мультипликативной группы GF(25)*. По условию $x$ — примитивный элемент, следовательно, GF(25) = $\langle x \rangle$ — циклическая группа порядка $n = 5^2 - 1 = 24$.

### Теоретическое обоснование
В циклической группе порядка $n$ порядок любого элемента $x^k$ вычисляется через наибольший общий делитель показателя и порядка группы:

$${ord}(x^k) = \frac{n}{\gcd(k, n)}$$

Поскольку $n = 24$, возможные значения порядка соответствуют множеству делителей числа 24: $\{1, 2, 3, 4, 6, 8, 12, 24\}$.

### Распределение элементов по порядкам
Вместо полной таблицы представим элементы, сгруппированные по их порядкам $d$, где количество элементов для каждого $d$ определяется функцией Эйлера $\varphi(d)$.

| Порядок $d$ | Количество элементов $\varphi(d)$ | Показатели $k$, для которых ${ord}(x^k) = d$ |
| :--- | :---: | :--- |
| **1** | 1 | 0 |
| **2** | 1 | 12 |
| **3** | 2 | 8, 16 |
| **4** | 2 | 6, 18 |
| **6** | 2 | 4, 20 |
| **8** | 4 | 3, 9, 15, 21 |
| **12** | 4 | 2, 10, 14, 22 |
| **24** | 8 | 1, 5, 7, 11, 13, 17, 19, 23 |

**Вывод:** Группа содержит 8 примитивных элементов (порядка 24). Элемент $x^0 = 1$ является единственным элементом порядка 1.

## Задача 2. Анализ (15, 4)-кода и LFSR

Дан проверочный полином $h(x) = x^4 + x + 1$ над $GF(2)$.

### 1. Динамика состояний регистра
Регистр реализует линейную рекурренту $a_{t+4} = a_{t+1} \oplus a_t$. Состояние регистра в момент $t$ обозначим вектором $S_t = (a_t, a_{t+1}, a_{t+2}, a_{t+3})$.

Начальное заполнение: $S_0 = (0, 0, 1, 1)$.

| $t$ | $S_t$ | Вычисляемый бит $a_{t+4}$ |
| :--- | :---: | :--- |
| 0 | 0011 | $0 \oplus 0 = 0$ |
| 1 | 0110 | $0 \oplus 1 = 1$ |
| 2 | 1101 | $1 \oplus 1 = 0$ |
| 3 | 1010 | $1 \oplus 0 = 1$ |
| 4 | 0101 | $0 \oplus 1 = 1$ |
| 5 | 1011 | $1 \oplus 0 = 1$ |
| 6 | 0111 | $0 \oplus 1 = 1$ |
| 7 | 1111 | $1 \oplus 1 = 0$ |
| 8 | 1110 | $1 \oplus 1 = 0$ |
| 9 | 1100 | $1 \oplus 1 = 0$ |
| 10 | 1000 | $1 \oplus 0 = 1$ |
| 11 | 0001 | $0 \oplus 0 = 0$ |
| 12 | 0010 | $0 \oplus 0 = 0$ |
| 13 | 0100 | $0 \oplus 1 = 1$ |
| 14 | 1001 | $1 \oplus 0 = 1$ |
| 15 | **0011** | (Возврат к $S_0$) |

### 2. Верификация кода максимальной длины
Длина последовательности до первого повторения составила $L = 15$. 
Для регистра размерности $m=4$ максимально возможный период равен $2^m - 1 = 2^4 - 1 = 15$.
Так как $L = 2^m - 1$, данный код является кодом максимальной длины (M-последовательностью), а полином $h(x)$ — примитивным.

### 3. Минимальное кодовое расстояние
Рассматриваемый $(15, 4)$-код является дуальным к коду Хэмминга, то есть симплексным кодом. 
Свойство симплексного кода: все ненулевые кодовые слова имеют одинаковый вес Хэмминга $w$.
$$d_{\min} = w = 2^{m-1}$$
Для $m = 4$:
$$d_{\min} = 2^{4-1} = 2^3 = 8$$

## Итоговые результаты
1. **Задача 1:** Порядки элементов определены через $\frac{24}{\gcd(k, 24)}$. Группа циклическая, структура распределения порядков приведена в итоговой таблице.
2. **Задача 2:** - Смена состояний прослежена до замыкания цикла (15 шагов).
   - Подтверждено свойство максимальной длины ($15 = 2^4 - 1$).
   - Минимальное расстояние кода $d_{\min} = 8$.
